Bienvenido a la API de Adopción de Mascotas

Esta API permite gestionar información sobre mascotas y cuidadores. Se puede acceder a endpoints para listar, buscar, filtrar, agregar, editar y eliminar datos.

Mascotas (Pets)

  • GET /api/pets

    Obtener todas las mascotas.

  • GET /api/pets/:id

    Obtener una mascota por ID.

  • GET /api/pets/search?name=Nina

    Buscar mascotas por nombre.

  • GET /api/pets/filter?type=gato

    Filtrar mascotas por tipo.

  • GET /api/pets/filter?sex=hembra

    Filtrar mascotas por sexo.

  • GET /api/pets/filter?type=gato&&sex=hembra

    Filtrar mascotas por tipo y sexo.

  • POST /api/pets

    Agregar una nueva mascota.

  • PUT /api/pets/:id

    Actualizar una mascota existente por ID.

  • DELETE /api/pets/:id

    Eliminar una mascota por ID.

Cuidadores (Caretakers)

  • GET /api/caretakers

    Obtener todos los cuidadores.

  • GET /api/caretakers/:id

    Obtener un cuidador por ID.

  • GET /api/caretakers/search?name=patitas

    Buscar cuidadores por nombre.

  • GET /api/caretakers/filter?type=refugio

    Filtrar cuidadores por tipo.

  • GET /api/caretakers/filter?acceptsVisits=true

    Filtrar cuidadores según si aceptan o no visitas.

  • GET /api/caretakers/filter?type=particular&&acceptsVisits=true

    Filtrar cuidadores por tipo y si aceptan visitas.

  • POST /api/caretakers

    Agregar un nuevo cuidador.

  • PUT /api/caretakers/:id

    Actualizar un cuidador existente por ID.

  • DELETE /api/caretakers/:id

    Eliminar un cuidador por ID.
